Mijn leven.

Ik ben Christian lemmen ik ben 16 jaar oud en ik zit op het mondriaan en ik volg de opleiding software developer. Ik heb all paar dingen gecodeerd zoals een hoger of lager game en een escape room en boter kaas en eiren. Ik heb all veel kennis in ict want op mijn middel bare school was ik heel vaak bezig op de computer want dat was mijn profiel vak. Ik zat alijd iets te doen met computers dus wanneer ik klaar was met mijn middelbare school was ik gelijk in het software gegaan want mijn droom is om een game developer te worden.
